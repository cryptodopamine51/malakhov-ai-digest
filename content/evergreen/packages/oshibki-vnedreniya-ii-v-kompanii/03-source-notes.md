| Утверждение | Источник | Tier |

## White-list использован

- Яков и Партнёры (Tier 1) — главный numerical anchor по эффекту.
- TAdviser, CNews Analytics — общий контекст кейсов российских компаний.
- Gartner — общий контекст PoC отказов.
- Роскомнадзор — primary для штрафов.
- Sber GigaChat — primary для российских тарифов.

## Не использовано (для прозрачности)

- Habr / vc.ru — есть отдельные посты про провалы, но конкретику без атрибуции брать рискованно.
- РБК Тренды — исключён из white-list.
