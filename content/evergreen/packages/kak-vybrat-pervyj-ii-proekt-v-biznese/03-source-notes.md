# Source Notes: Как выбрать первый ИИ-проект в бизнесе

Проверено 22 мая 2026 года.

| Утверждение | Источник | Tier |
|---|---|---|
| «9 из 10 успешных российских внедрений ИИ начинают с одного узкого процесса» | Яков и Партнёры, отчёт 2025 | Tier 1 |
| Тарифы GigaChat / YandexGPT | developers.sber.ru / aistudio.yandex.ru | Primary |
| 30% gen-AI PoC abandoned (Gartner July 2024) | Gartner press release | Primary international |
| Кейс дистрибьютора (CRM Bitrix24, 240 КП за 45 дней) | Редакционный пример | Editorial |
| Worked example скоринга для отзывов | Расчёт по формуле | Editorial |
| Бюджет первого пилота 80–350 тыс ₽ | Сводка по российским внедрениям из TAdviser/CNews | Tier 1 (агрегация) |

## White-list использован

- Яков и Партнёры — статистический anchor.
- TAdviser, CNews Analytics — бюджетные ориентиры.
- Sber GigaChat / Yandex AI Studio — тарифы для скоринга.
- Gartner — общий контекст.

## Editorial markers

- Кейс дистрибьютора и worked example отзывов — оба помечены «Редакционный пример».
- Скоринговая система — авторская, не из публичного источника. Числа весов обоснованы практикой Malakhov AI и могут пересматриваться в будущих ревизиях гайда.
