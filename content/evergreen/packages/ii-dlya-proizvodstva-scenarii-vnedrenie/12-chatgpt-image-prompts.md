## Технический контекст

Не отправляйте этот раздел в ChatGPT.

- Финальные WebP-пути уже прописаны в `08-metadata.json`.
- Команда после сохранения PNG:

```bash
npm run images:prep -- --slug=ii-dlya-proizvodstva-scenarii-vnedrenie
```

- Скрипт создаёт canonical WebP 1200×675 для cover, 1200×800 для inline, плюс `-480.webp` и `-768.webp`.
- Если ChatGPT отдаст случайные имена файлов, это допустимо: `images:prep` сматчит их по порядку слотов из metadata. Но точные имена из `Save As` снижают риск ошибки.
- После конвертации нужно запустить:

```bash
npm run evergreen:check -- --slug=ii-dlya-proizvodstva-scenarii-vnedrenie
```
