# SERP Research: Какие бизнес-процессы автоматизировать с помощью ИИ

## Анализ топа SERP по запросу «какие процессы автоматизировать с помощью ИИ» (по состоянию 22 мая 2026)

Что встречается у большинства материалов конкурентов:

- общий список «нейросеть умеет писать тексты, отвечать в чате, делать картинки» — без привязки к процессу;
- ссылки на западные кейсы McKinsey/BCG без локализации под Россию;
- отсутствие критериев отбора процесса;
- отсутствие counter-strategy («когда не стоит»);
- отсутствие worked example с конкретной окупаемостью.

Что не закрывает топ:

- матрица из 4 фильтров (эффект, сложность, риск, данные) для самостоятельной оценки процесса;
- развёрнутый кейс с метриками pre-sales по российскому/СНГ-сегменту;
- numerical worked example с двумя сценариями автоматизации (65% / 40%);
- регуляторный аспект 152-ФЗ и спецкатегорий персональных данных в HR-сценариях;
- честные стоп-сигналы вместо «возможностей ИИ».

## Уникальный угол Malakhov AI

- Матрица 4 фильтров как тестируемый чек-лист.
- Чёткое разделение «🟢 первый кандидат / 🟡 второй этап / 🔴 откладываем».
- Локальный регуляторный контекст в HR и финансах.
- Counter-strategy 7 критериев — больше, чем у большинства материалов.
