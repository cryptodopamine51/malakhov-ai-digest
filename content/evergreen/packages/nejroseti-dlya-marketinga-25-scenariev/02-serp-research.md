# SERP and Reference Research: Нейросети для маркетинга: 25 сценариев

Checked date: 2026-06-12 (редакционный анализ интента без живого SERP-прогона)

## Типовая выдача по запросу «нейросети для маркетинга»

1. **Каталоги инструментов** («30 нейросетей для маркетолога») — списки сервисов с описанием в одну строку, без процесса и экономики.
2. **Блоги сервисов рассылок/CRM** — 5–10 сценариев на уровне «пишите посты быстрее», с промо собственного продукта.
3. **Курсы и инфобиз** — обещания «маркетолог 10x», без ограничений и рисков.

## Повторяющиеся блоки

- Список инструментов, примеры промптов, «плюсы и минусы», базовые сценарии контента.

## Чего не хватает (наш дифференциатор)

- Группировка по процессу маркетинга, а не по типам инструментов.
- Экономика с расчётом и порогом «когда в минус».
- Регуляторика: маркировка ИИ-креативов (erid/ОРД), 152-ФЗ для клиентских данных.
- Честные ограничения: прогнозы спроса, конкурентный анализ «по самоописанию», карточки с выдуманными свойствами.
- Регламент и владелец процесса как условие эффекта.

## Решение по углу

Карта 25 сценариев в 6 направлениях со зрелостью групп + кейс + worked example + counter-strategy. Инструменты упоминаются внутри сценариев, отдельного каталога нет (anti-staleness).
