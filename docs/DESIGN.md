## Концепция

Editorial минимализм в духе качественных научных изданий (NYT Science, MIT Tech Review).
Дизайн строится на контрасте: белый фон, чёрная типографика, живые фотографии.
Главный визуальный ход — wireframe-иллюстрации, вписанные в заголовки секций (для AI: схемы нейросетей, процессоры, абстрактные геометрические структуры).

**Ключевые принципы:**
- Типографика первична — контент важнее декора
- Меньше скруглений — острые углы создают серьёзный, редакционный характер
- Один акцентный цвет, использовать экономно
- Читаемость на русском языке — приоритет

## Типографика

### Шрифты

| Применение | Шрифт | Вес | CSS-переменная |
|---|---|---|---|
| Заголовки H1–H2, display | **Onest** (geometric sans) | 700–900 | `--font-onest` / `font-serif` |
| Body text, UI, навигация | **Golos Text** (humanist sans) | 400–600 | `--font-golos` / `font-sans` |
| Данные, числа, метки, kicker | **IBM Plex Mono** (monospace) | 400–700 | `--font-mono` / `font-mono` |
| Pull-quote | Onest bold | 700 | — |

> **Onest** — создан для цифровых продуктов, отличная кириллица, геометрический гротеск. Используется в tech-компаниях СНГ. Заменил Playfair Display (газетный serif не соответствовал AI/tech вектору издания).
> **Golos Text** — российский шрифт, превосходная читаемость русского текста на любых размерах.
> **IBM Plex Mono** — моноширинный шрифт IBM, задаёт технический характер числам и данным в редакционных блоках.

**Правило использования mono:** все числовые значения в статблоках, kicker-подписи, метаданные источника/даты в нижних колонтитулах, метки категорий в редакционных блоках — рендерятся `font-mono`.

### Размеры (desktop)

| Элемент | Размер | Line-height |
|---|---|---|
| H1 главная (hero) | 80–120px | 1.0 |
| H1 страница категории | 48px | 1.1 |
| H1 статья | 40–48px | 1.15 |
| H2 секция | 28px | 1.2 |
| H3 карточка | 18–20px | 1.3 |
| Body text | 17px | 1.65 |
| Лид статьи | 20px | 1.5 |
| Навигация | 12px uppercase | — |
| Теги | 11px uppercase | — |
| Метаданные (источник, дата) | 13px | — |

### Правила читаемости

- Body text `max-width: 640px` — критично для русского языка
- `letter-spacing: 0.06em` для uppercase элементов (теги, навигация)
- Заголовки карточек: `-webkit-line-clamp: 3` — обрезать после 3 строк

## Компоненты

### Хедер

```
[ЛОГО]    Индустрия  Исследования  Лаборатории  Россия  Код    [🔍] [☀/🌙] [☰]
```

- Фиксированный (`position: sticky top-0`)
- Фон: `rgba(255,255,255,0.95)` + `backdrop-filter: blur(8px)`
- Нижняя линия `1px solid #E5E5E5` появляется при скролле
- Лого: жирный гротеск, ~18px, буквы логотипа
- Категории: Golos Text, 12px, uppercase, `letter-spacing: 0.08em`
- Переключатель темы: иконка 20×20, между поиском и бургером
- Мобайл: лого + иконки (поиск, тема, бургер)

### Карточка статьи (стандарт)

```
[Изображение 16:9, border-radius: 4px]
[ТЕГ] [ТЕГ]                        ← outlined, 11px uppercase, border-radius: 3px
Заголовок жирным, 3 строки max
Источник [favicon]          Дата    ← #888888, 13px
```

- Hover: `transform: translateY(-2px)`, тень `0 4px 16px rgba(0,0,0,0.08)`
- Заголовок hover: цвет → акцент `#0055FF`
- Нет box-shadow в обычном состоянии

### Featured карточка (главная в секции)

- Изображение на полную ширину карточки
- Gradient overlay снизу: `linear-gradient(transparent, rgba(0,0,0,0.75))`
- Теги + заголовок + "Читать" кнопка — поверх изображения, белым текстом
- Кнопка "ЧИТАТЬ": `border: 1.5px solid #FFFFFF`, `border-radius: 4px`, белый текст, прозрачный фон

### Теги / бейджи

- Outlined: `border: 1px solid currentColor`, `border-radius: 3px`, padding `3px 8px`
- Активный / selected: чёрный фон, белый текст
- Специальный "НОВОЕ": акцентный цвет `#0055FF`, белый текст
- Горизонтальный scroll-ряд тегов: скрыть scrollbar, но оставить функциональность

### Индикатор важности статьи

Для статей с высоким score (ТОП ДНЯ):
- Тонкая левая полоска `3px solid #0055FF` у карточки
- Или бейдж "ТОП" акцентного цвета рядом с тегами

### Кнопки

| Тип | Стиль |
|---|---|
| Primary (READ MORE) | `bg: #111`, `color: #fff`, `border-radius: 4px`, padding `10px 20px` |
| Secondary (SEE ALL) | `border: 1.5px solid #111`, `bg: transparent`, `border-radius: 4px` |
| Мобайл SEE ALL | полная ширина, `bg: #111`, `color: #fff` |
| Стрелка (next/prev) | `border: 1px solid #E5E5E5`, квадратная 40×40px |

## Страницы

### Главная `/`

1. **Hero-блок** в тонкой рамке:
   - Название издания гигантским serif (~120px)
   - Поверх — wireframe-иллюстрация AI-тематики (нейросеть, схема чипа)
   - Под ним — 2 равные карточки с изображением

2. **Лента по секциям** (для каждой категории):
   - Заголовок категории + кнопка "Все новости →"
   - Горизонтальный ряд тегов-фильтров
   - Сетка: 1 featured (центр) + 1 малая слева + 2–3 текстовых строки справа

### Страница категории `/topics/[slug]`

- Hero: название категории крупным + wireframe-иллюстрация, всё в рамке
- Под ним: featured статья на полную ширину
- Далее: сетка 3 колонки
- Пагинация или infinite scroll

### Страница статьи `/articles/[slug]`

**Desktop:**
- Хедер-блок в рамке: wireframe + теги + H1 + автор/дата
- Двухколоночный layout: левая 25% (источник, курсивная заметка) + правая 70% (текст)
- Лид: 20px, жирный
- Pull-quote: курсив, тонкая рамка слева `3px solid #E5E5E5`
- Изображения: в рамке `1px solid #E5E5E5`, caption под ним серым
- В конце: шеринг + ← НАЗАД / ВПЕРЁД →
- "Похожие статьи" — 4 карточки в ряд
- "Популярное в категории" — mixed layout

**Мобайл:**
- Иллюстрация/фото полной ширины сверху
- Теги, H1, автор/дата
- Единая колонка, `padding: 0 16px`

**UX-улучшения:**
- Прогресс-бар чтения: `4px` линия под хедером, акцентный цвет
- Sticky хедер показывает заголовок статьи при скролле вниз
- Время чтения рядом с датой: "5 мин"

### Страница источников `/sources`

- Карточки источников: логотип + название + описание + количество статей + последняя дата
- При клике — фильтрованный список статей этого источника

## Футер

```
[Подписка на email рассылку]

[FB] [TG] [RSS] [YT]

Навигация (3 колонки)    Категории

```

- Фон: `#0A0A0A` (в обеих темах одинаковый)
- Telegram CTA — выделенный блок с акцентом
- Email input: `border-radius: 4px`

## Мобильные паттерны

- Навигация: бургер → slide-down панель со всеми разделами
- Featured: полноширинное фото + overlay снизу с градиентом
- Список: статьи разделены `1px solid #E5E5E5` (не отдельные карточки — экономит место)
- "SEE ALL" — полная ширина, `border-radius: 4px`
- Swipe-горизонталь для тегов-фильтров

---

## Иконки и иллюстрации

- Иконки: Lucide React (линейные, stroke-based, 20×20)
- Wireframe-иллюстрации для hero-блоков: SVG, чёрные линии на белом (`stroke: #111`, `fill: none`)
  - Главная: абстрактная нейронная сеть или процессор
  - Индустрия: схема бизнес-процесса
  - Исследования: молекула или граф
  - Лаборатории: лабораторная колба / формула
  - Россия: карта-контур
  - Код: терминал / bracket схема
- Фавиконки источников: загружаются через `favicon.im` или хранятся локально
