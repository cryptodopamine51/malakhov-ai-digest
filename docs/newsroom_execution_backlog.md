## Определение Готового Продукта

Продукт считается доведенным до рабочего состояния, когда:

- главная показывает сильную витрину
- слабые материалы не попадают в публичный слой
- статьи читаются как редакционный текст
- весь публичный слой понятен на русском языке
- страницы пригодны для поисковой выдачи
- у материалов есть устойчивый визуальный слой
- редакция может вручную усиливать ключевые публикации
