## Блок B · Главная страница в стиле VC.ru

### B1 · Секция "Топ сегодня"
В `app/page.tsx`:
- Верхний блок: 5–7 статей с наивысшим score за последние 24ч (МСК)
- Горизонтальный скролл на мобильном / 2-3 колонки на десктопе
- Визуально выделен: заголовок "Топ за сегодня" с датой
- Отдельный запрос в `lib/articles.ts`: `getTopTodayArticles(limit: 7)`

### B2 · Лента по популярности с пагинацией
В `app/page.tsx` ниже топ-блока:
- Заголовок "Все новости"
- Карточки 3-колонками, сортировка по score DESC
- Кнопка "Загрузить ещё" — подгружает следующие 12 статей (Client Component)
- Либо страничная пагинация: `/page/2`, `/page/3` (лучше для SEO)

### B3 · Score с учётом свежести
В `pipeline/scorer.ts` или `lib/articles.ts`:
- Формула: `effective_score = score × freshness_multiplier`
- `freshness_multiplier`: 1.0 если < 24ч, 0.8 если 24–48ч, 0.6 если 48–72ч, 0.4 если > 72ч
- Применяется при выдаче ленты, не меняет поле score в БД
