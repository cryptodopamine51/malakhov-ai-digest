# Malakhov AI Digest — Visual Style Guide v2
## Updated after first generation review

## 1. Purpose

This document defines the **second-generation visual system** for article illustrations in **Malakhov AI Digest**.

This version updates the original guide after testing the first prompt batch.  
The goal now is not only to preserve a recognizable style, but also to **prevent early template repetition** and push the visuals closer to a true **editorial magazine collage language**.

The system must achieve:

1. **Recognizable publication identity**
2. **High variety across articles**
3. **Strong editorial metaphor**
4. **Low “AI business template” feel**
5. **More photomontage and magazine-cover tension**

---

## 2. What the first generation proved

The first generation already established a useful base:

- restrained palette
- matte paper texture
- calm editorial tone
- clear AI/news relevance
- readable concepts

But it also revealed recurring shortcuts that will become repetitive if left uncontrolled:

- glowing network spheres
- floating dashboard tiles
- centered business figures
- beige symmetrical layouts
- repeated “person + interface panels” composition
- too much literal analytics UI

This v2 guide corrects for that.

---

## 3. Updated Core Style Definition

**Malakhov AI Digest visual language** is a **premium editorial conceptual photomontage system** with a tactile magazine-cover character.

It should feel like:

- a serious publication about AI, power, media, business, and society
- a conceptual editorial illustration, not a product graphic
- a paper-based collage world with layered symbolic thinking
- varied from article to article, but unmistakably part of one publication

### Style essence
- editorial photomontage
- conceptual collage
- print-era media aesthetics
- modernist poster influence
- magazine-cover composition
- tactile paper surface
- restrained color intelligence
- symbolic clarity

### What the image should communicate
- one idea
- one visual center
- one strong metaphor
- one memorable editorial impression

The system drifted slightly toward:

The system should shift toward:

## 5. Fixed Style Invariants

These should remain stable across the series.

## 5.1 General identity
- premium editorial illustration
- idea-driven, not decorative
- suitable for a serious AI publication
- not a stock-business image
- not glossy corporate art
- not a futuristic neon cliché

## 5.2 Composition
- one clear focal point
- 1 to 3 dominant visual masses
- negative space is important
- readable silhouette
- quick conceptual readability within 2–3 seconds
- strong cover-like balance

## 5.3 Visual language
- layered editorial photomontage
- realistic subjects mixed with graphic abstraction
- paper-collage feel
- symbolic objects
- architecture, documents, screens, public institutions, media artifacts
- large modernist shapes: circles, blocks, panels, arcs, silhouettes, poster planes

## 5.4 Surface and texture
- matte paper feel
- subtle print grain
- tactile surface
- soft halftone / print residue
- slight vintage editorial wear
- no glossy CGI finish

## 5.5 Palette behavior
Keep the palette restrained and publication-like.

Base palette directions:
- coral / salmon
- mustard / ochre / warm yellow
- teal / dusty turquoise
- dusty blue
- graphite / charcoal / deep navy
- off-white / paper gray / warm beige

Palette rule:
- one warm accent
- one cool counterbalance
- one paper-like base
- one dark anchor

---

## 6. New Editorial Priority Rules

These rules are stronger than in v1.

## 6.1 Prefer editorial metaphor over literal explanation
The image should not try to explain the topic with too many obvious UI blocks or generic chart panels.

Prefer:
- symbolic tension
- editorial abstraction
- cultural cues
- public or institutional mood
- media objects
- stronger conceptual framing

## 6.2 Prefer photomontage over pure digital illustration
The image should feel assembled from layered visual ideas, not rendered as one smooth generic scene.

## 6.3 Prefer composition personality over symmetry
The image should often feel like a designed magazine cover, not a centered template.

---

## 7. What Must Change Between Articles

These variables must rotate regularly:

- subject type
- presence or absence of people
- scene type
- metaphor type
- scale relationships
- camera distance
- dominant shape logic
- amount of architecture
- amount of media objects
- level of abstraction
- degree of asymmetry
- amount of interface material

This is essential:
**the publication should feel unified, but never repetitive.**

---

## 8. Anti-Template Rules

This is now a core section of the system.

## 8.1 Avoid overused series shortcuts
Do not overuse:
- glowing network spheres
- floating dashboard tile clusters
- centered business people
- browser-window stacks
- repeated beige symmetrical layouts
- generic “person at laptop” scenes
- chart-heavy explanation panels

## 8.2 Rotate away from easy AI-business defaults
The system should actively rotate toward:
- cropped portraits
- architecture
- documents
- newspapers
- cameras
- public institutions
- editorial rooms
- symbolic objects
- poster-like graphic masses
- conceptual cover compositions

## 8.3 Use interface fragments sparingly
Interfaces can appear, but should not dominate most of the series.

Use them as accents, not as the default backbone.

---

## 9. Composition Modes

Rotate across these modes deliberately.

## Mode 1. Human + System
Use for:
- tools
- agents
- AI in work
- adoption stories
- product launches
- operational AI

Ingredients:
- one human figure
- one system layer
- symbolic relation between human and structure

Important:
Avoid turning this into a repetitive “businessman with laptop and panels” pattern.

Variation ideas:
- figure from behind
- cropped close profile
- person inside a larger system space
- person facing an abstract structure
- person as a small figure in relation to a giant intelligence form

---

## Mode 2. Object + Symbol
Use for:
- chips
- security
- infrastructure
- identity
- data
- trust
- capital
- strategy

Ingredients:
- one strong central object
- one supporting symbolic layer
- strong cover logic
- minimal but memorable composition

Examples:
- shield + legal paper
- camera + fractured identity
- globe + institutional node pattern
- archive box + network lines
- microphone + signal grid

---

## Mode 3. Social / Editorial Scene
Use for:
- newsroom stories
- teams
- institutions
- editorial processes
- adoption and collaboration

Ingredients:
- 3–8 people
- media or work environment
- printed materials, screens, pages, layouts, documents
- symbolic information flow

Important:
This should feel like a serious editorial or institutional situation, not a generic office brochure.

---

## Mode 4. Architecture + Technology
Use for:
- regulation
- state policy
- geopolitics
- public AI infrastructure
- national strategy
- governance

Ingredients:
- parliament, capitol, skyline, public building, civic façade, city mass
- one technological or informational layer
- one symbolic legal or structural element

This mode should carry strong institutional weight.

---

## Mode 5. Interface + Metaphor
Use for:
- product news
- software
- agent workflows
- automation tools
- coding tools
- platforms

Ingredients:
- interface fragment
- symbolic metaphor
- clean but editorial composition

Important:
Use charts and dashboards sparingly.
The image should still feel like media illustration, not a SaaS product page.

---

## Mode 6. Abstract Concept Cover
Use for:
- deepfakes
- AGI
- alignment
- opacity
- AI safety
- cognition
- identity
- future-of-AI essays

Ingredients:
- portrait fragment, mask, statue, visual split, abstract construction, media surface
- fewer literal environment cues
- more tension, symbolism, and cover-level concept clarity

This mode should be one of the strongest brand assets of the publication.

---

## 10. Recommended Series Mix

To keep the publication visually healthy, use approximate rotation targets over a batch of images.

- 25% Architecture + Technology
- 20% Abstract Concept Cover
- 20% Social / Editorial Scene
- 20% Human + System
- 15% Object + Symbol

This prevents the series from becoming dominated by one easy composition.

## 12. Stronger Editorial Photomontage Rules

This section is new and important.

The image should feel more like:
- a designed editorial cover
- a layered media illustration
- a print-era conceptual collage

And less like:
- a clean AI platform graphic
- a dashboard-centered digital illustration
- a generic business technology hero shot

### Desired photomontage characteristics
- layered paper logic
- cutout-like spatial relationships
- mixed scales
- visible tension between objects
- unexpected but controlled juxtapositions
- art-directed asymmetry
- cover-level visual confidence

---

## 13. Interface Discipline Rules

Interfaces are allowed, but must be controlled.

## 13.1 Use interfaces as fragments
Use:
- partial browser windows
- cropped dashboard areas
- a single chart fragment
- UI surfaces as visual texture

## 13.2 Do not make most illustrations dashboard explanations
Avoid turning article covers into:
- report screens
- control panels
- multi-widget analytics boards
- product-demo graphics

## 13.3 Prefer symbolic editorial objects over literal dashboards
Prefer:
- documents
- newspapers
- camera frames
- legal pages
- maps
- public architecture
- objects of trust / identity / control / communication

---

## 14. Asymmetry and Layout Discipline

To keep the series from feeling templated:

- not every focal point should be centered
- large empty areas are welcome
- cropped elements can add tension
- figures can be off-center
- big masses can live to one side
- depth can be flatter or more poster-like
- some covers should feel closer to print poster logic than to symmetric digital artwork

---

## 15. Quality Criteria

Every final image should pass these questions.

## Editorial clarity
- Is the main article theme readable quickly?
- Is there one clear visual center?
- Is the metaphor strong enough?

## Publication consistency
- Does it feel like part of Malakhov AI Digest?
- Does it match the series palette and tactile tone?

## Originality
- Does it avoid repeating the easy series shortcuts?
- Does it bring a fresh composition or metaphor?

## Editorial character
- Does it feel like a serious magazine cover?
- Does it carry enough cultural, public, or symbolic weight?

## Restraint
- Is the palette controlled?
- Are the interface elements limited?
- Does the composition feel clean and intentional?

---

## 16. Negative Rules

Avoid these directions unless you intentionally want an experimental branch outside the main publication language.

- generic corporate vector illustration
- smiling startup brochure scenes
- glossy futuristic AI brain clichés
- neon cyberpunk visual language
- hyper-detailed sci-fi renders
- overexplained dashboard compositions
- repetitive laptop-and-tiles layouts
- visual clutter with weak hierarchy
- shallow “tech but clean” SaaS aesthetics

## 22.1 Article-specific prompt template

```text
Article topic: [TOPIC]
Core message: [WHAT THIS NEWS MEANS]
Main metaphor: [METAPHOR]
Preferred scene type: [HUMAN + SYSTEM / OBJECT + SYMBOL / SOCIAL SCENE / ARCHITECTURE + TECHNOLOGY / INTERFACE + METAPHOR / ABSTRACT CONCEPT COVER]
Key visual elements: [ELEMENTS]
Mood: [CALM / TENSE / STRATEGIC / INVESTIGATIVE / GLOBAL / INSTITUTIONAL / ANALYTICAL]
Complexity level: [LOW / MEDIUM / HIGH]
Asymmetry level: [LOW / MEDIUM / HIGH]
Experiment level: [10% / 20% / 30%]
```

```text
Create an original editorial conceptual collage illustration for an AI news digest article.
```

## 24. Practical Review Checklist

Before approving an image, check:

- Does it look too much like a previous image?
- Does it overuse dashboards or tiles?
- Does it feel more like a magazine than a software ad?
- Is there a strong metaphor?
- Is the composition distinctive enough?
- Is the tactile paper/editorial quality present?
- Does the subject fit the correct scene mode?

If several answers are weak, regenerate.
