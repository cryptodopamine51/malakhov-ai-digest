# Генератор отмазок

Одностраничный статический веб-сервис без внешнего API. Пользователь выбирает одну из 4 рабочих ситуаций, тон и уровень смелости, получает готовую отмазку, рейтинг правдоподобности и кнопку копирования. На максимальной смелости генератор переходит в грубый офисный режим с прямым матом.

## Локальный запуск

```bash
cd excuse-generator
npm run check
npm run dev
```

Страница будет доступна на `http://localhost:4173`.

## Деплой

Проект рассчитан на Vercel как статический сайт:

```bash
cd excuse-generator
vercel --prod --yes
```
